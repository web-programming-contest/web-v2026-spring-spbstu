Задание 5. Реализовать механику компонента “подробнее”

Компонент аккордеон для отображения часто задаваемых вопросов (FAQ).
Функциональность:
- открытие/закрытие отдельных элементов;
- анимации плавного раскрытия;
- возможность открывать один или несколько элементов одновременно.

Примечание: библиотеки использовать не рекомендуется. При наличии согласовывать их использование с преподавателями.
