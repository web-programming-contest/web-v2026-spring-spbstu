Задача 5. Реализовать механику списка в книжном интернет-магазине
Создать компонент списка в виде дерева (tree). Добавить возможность
скрывать/раскрывать ветки.
Дизайн дерева остается за студентом.
Структура предполагаемого дерева (по уровням):
- Автор;
 - Издательство;
 - Жанр;
 - Название книги.
