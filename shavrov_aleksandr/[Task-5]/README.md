# Лабораторная работа 5 — React

**Вариант 34**

## Задание

Задача 5. Создать приложение прогноза погоды
Приложение будет отображать текущую погоду и прогноз на несколько
дней вперед с использованием открытого API погодных данных.
Функциональность:
- текущая погода с основными показателями;
- возможность сменить город;
- возможность менять интервал просмотра погоды (1 день, 3 дня, 7 дней)
– опционально.
