<h1>Вариант 11</h1>  
<h2>Задача 5. Создать шахматную игру</h2>  
Создайте интерактивную шахматную игру с возможностью игры против игрока (можно сделать против компьютера). Проект должен включать шахматную доску с фигурами, реализацию основных правил и визуальную обратную связь при ходе.  

<b>Функциональность:</b>

- отображение доски с чередующимися светлыми и темными клетками;
- перемещение фигур по правилам шахмат;
- определение конца игры (шах и мат);
- подсветка доступных ходов при выборе фигуры.

<b>Логика игры:</b>

- поочередные ходы игроков;
- запрет ходов, оставляющих своего короля под шахом;
- автоматическое определение возможных ходов для каждой фигуры.
