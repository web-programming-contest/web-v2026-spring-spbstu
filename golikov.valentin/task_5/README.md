**Задача 5. Реализовать механику рейтинга в книжном интернет-магазине**

Создать список книг, который будет содержать обложку книги, название, рейтинг, описание и цену.

В каждом компоненте книги добавить возможность выставлять рейтинг книги пользователем.

Рейтинг рассчитывается по средней оценке (сумма оценок деленное на их количество).

Функционал компонента рейтинга:
- принимает количество звезд (по умолчанию 5);
- принимает рейтинг (число);
- возможность изменить рейтинг пользователем (после голосования, на 3-4 с. после оценки отображается выбранный пользователем, затем возвращается в среднее состояние).
