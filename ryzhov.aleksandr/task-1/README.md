Сверстать флаг Бангладеша, сам красный круг сделать в центре с помощью position: absolute.

Я решил сделать флаг в соответствие с описанными на Википедии пропорциями (чуть сложнее, но интереснее):
ширина/высота = 5/3, 
радиус круга = 1/5 ширины, 
положение круга - 9/20 ширины.

Так как размер окна может изменяться, привязал расположение и размер круга к ширине, по высоте выровнял по центру.
