# Вариант 22
## Задача 2 
Реализовать функцию, которая кодирует строку с использованием алгоритма RLE (Run-Length Encoding: "AAABBBCC" -> "3A3B2C").
### Тестовые значения 
1. Пустая строка
"" ---> ""
2. Один символ
"a" ---> "1a"
3. Все символы разные
"abcde" ---> "abcde"
4. Все символы одинаковые
"aaaaa" ---> "5a"
5. Чередующиеся символы
"aabbbccccddddd" ---> "2a3b4c5d"
